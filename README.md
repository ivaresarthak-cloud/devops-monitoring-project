# 🚀 DevOps Monitoring Project

## 📌 Project Overview
This project demonstrates a complete DevOps workflow where a Python application is containerized using Docker, deployed on Kubernetes, and monitored using Prometheus and Grafana on AWS.

---

## 🛠️ Tech Stack
- Docker
- Kubernetes (Minikube)
- AWS EC2
- Prometheus
- Grafana
- Python (Flask)

---

## Features
- Dockerized application
- Kubernetes deployment
- NodePort service exposure
- Monitoring with Prometheus
- Visualization with Grafana
